The benchmarks in this directory were generated with [DDVerify](http://www.cprover.org/ddverify/), which is described in the paper

Thomas Witkowski, Nicolas Blanc, Daniel Kroening, and Georg Weissenbacher.
Model Checking Concurrent Linux Device Drivers.
Proc. ASE, pages 501-504, 2007. ACM.

There is one separate file for each assertion in file ddv_machzwd_all.i
(i.e., _all contains all claims).
The other postfixes indicate the name of the function in which the
assertion occurs.

*.i   The files have been (slightly) pre-processed with CIL.
