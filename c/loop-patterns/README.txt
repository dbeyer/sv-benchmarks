Benchmarks used for computing Arithmetic progression, Geometric progression , Harmonic progression and related proerties. 
Contributed by : Shivayu Satija, Tata Consultancy Services, INDIA.

Benchmark Description :

1) Loop1 - Loop6 programs : Various series computation of A.P. and G.P. has been done, upto nth term. Property is checked, whether computed sum is satisfying the checks. All true programs.
 
3) Loop7 - Loop12 programs : Computation of sum of harmonic series, upto nth term. Property is checked, whether computed sum is satisfying the checks. All true programs.
